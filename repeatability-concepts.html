Repeatability: Four Visual Concepts

Each concept shows the same mock data: 10 runs produced 8 distinct findings. Some appeared in every run, some in most, some in only one or two. The question: which visual helps you grok the whole picture fastest?

1 · Heat Map

One row per finding. One column per run. Lit = appeared. Your eye reads solid rows as bedrock, patchy rows as uncertain, and single dots as noise.

Center = appeared in every run. Each ring outward = less frequent. The size of the core relative to the whole tells you how stable the review is.

Design Principles
✅ Blind First
All models analyze with NO role. Prevents persona distortion.
✅ Open Field
No pre-listed options. Models discover independently.
✅ Consensus ≠ Truth
Agreements are attacked, not rubber-stamped. Shared bias is the silent killer.
✅ Chair Separation
Context-blind for framing (Phase 2). Context-aware for synthesis (Phase 5).
Anti-Patterns (Learned)
✗ Roles before analysis → persona distortion
✗ Listing options → anchoring bias
✗ Only testing disagreements → shared hallucinations pass
✗ Chair with user context during comparison → biased framing
✗ Floating Chair model → quality fluctuates with session
